## 2024-05-23 - Logic-First Templates
**Learning:** UX in this repo means aligning contribution templates with the "Logic-First Framework" to reduce friction and improve argument quality. Standard UX (colors, buttons) is replaced by Process UX (guiding users to think logically via placeholders).
**Action:** When updating templates, explicitly map fields to logical components (Premises, Conclusion, Verifiability) and use placeholders to model the expected thought process.
