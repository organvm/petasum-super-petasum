## 2024-05-23 - GitHub Issue Templates as UX
**Learning:** UX isn't just about pixels; in a governance repo, the "user interface" is the issue template. By aligning the template fields directly with the governance philosophy (Logic-First), we reduce cognitive load and guide users to valid contributions without them needing to memorize the manual.
**Action:** When working on non-code repos, treat configuration files (like YAML templates) as the primary UI surfaces to be optimized for clarity and guidance.
